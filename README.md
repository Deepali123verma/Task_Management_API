# Task Management REST API

A **secure and scalable Task Management REST API** built with **Flask** and **PostgreSQL**, featuring **JWT authentication**, **role-based access**, **RSA encryption**, and **Swagger API documentation**. Fully **Dockerized** for easy deployment.

---

## 🚀 Features
- **User Authentication:** JWT-based secure login  
- **Role-Based Access:** Admin & User privileges  
- **Task Management:** Create, assign, update, delete, and track tasks  
- **Data Security:** RSA asymmetric encryption for sensitive data  
- **API Documentation:** Interactive Swagger UI  
- **Docker Support:** Easy containerized deployment  

---

## 🛠 Tech Stack
- **Backend:** Flask (Python)  
- **Database:** PostgreSQL  
- **Authentication:** JWT (RS256)  
- **API Docs:** Swagger (Flasgger)  
- **Deployment:** Docker, Railway
