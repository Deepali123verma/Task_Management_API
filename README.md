# 📝 Task Management REST API

A secure and scalable **Task Management System** built with **Flask (Python)** and **PostgreSQL**, featuring **JWT-based authentication**, user login, and task operations like add, update, delete, and 
view.
## 🚀 Features
- 🔐 JWT Authentication (Login API)
- 👤 User Management
- 📋 Task Creation & Assignment
- ✏️ Task Update and Delete
- 🗄️ PostgreSQL Integration
- 📡 RESTful API Architecture
## 🛠️ Tech Stack

- **Backend**: Python, Flask
- **Database**: PostgreSQL
- **Authentication**: Flask-JWT-Extended
- **Others**: Flask-CORS, psycopg2
